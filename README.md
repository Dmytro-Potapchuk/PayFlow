# PayFlow Frontend

Frontend application for the PayFlow project built with React Native and Expo.

## Technologies

- React Native
- Expo
- TypeScript
- React Navigation
- REST API

## Installation

Clone the repository

git clone https://github.com/Dmytro-Potapchuk/PayFlow.git

Go to project folder

cd PayFlow

Install dependencies

npm install

## Running the app

Start Expo development server

npx expo start

Then run on:

- Android emulator
- iOS simulator
- Expo Go mobile app
